# Welcome to the Flow Rules Repository

This repository contains the official **Notabene Flow Rules** — the published requirements, standards, and policies that create and govern the multi-lateral framework among all Partners in Notabene Flow.

---

## Current Status

> **⚠️ Draft**
>
> These rules are currently in **draft form** and under active development. Content is subject to change as we refine the governance framework, gather feedback, and prepare for the official launch.

---

## What to Expect

Once complete, this repository will provide:

- **Markdown-based rules** — All rules will be written in clean, readable markdown for easy navigation and version control

- **Click-through structure** — Rules will be organized with clear cross-references, allowing you to navigate seamlessly between related sections and definitions

- **Organized folder structure** — Rules will be separated into logical directories reflecting the tiered governance structure:
  - Tier 1: Constitutional Rules
  - Tier 2: Supermajority Rules  
  - Tier 3: Simple Majority Rules
